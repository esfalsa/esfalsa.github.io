Applesauce

Applesauce is a simple but versatile endorsement tool for NationStates that provides a single button to endorse multiple nations, one at a time, from a single tab.

I originally began developing Applesauce in February 2022 in order to streamline my endotarting process. The tools I was familiar with — the North Pacific’s endorsement tools and AugustinAndroid — all required repeatedly opening and closing tabs instead of just, well, endorsing.

That said, nowadays I primarily use Applesauce for military gameplay. Other tools, like Archer, offer the one-tab endorsement functionality without the need to download and install any code. Instead, it’s the cross-endorsement shortcuts, rather than endotarting functionality, that I find the most valuable in my personal usage.

Applesauce is free and open-source software released under the AGPL-3.0 license.

SPSF Data Portal

The SPSF Data Portal is a spreadsheet that offers a public database of, and statistics about, the operations of the South Pacific Special Forces. On the technical side of things, it’s just a Google Sheets spreadsheet that parses update reports and compiles the data.

I developed and introduced the portal in March and April 2022, supplanting earlier SPSF report parsers developed by HumanSanity and Luca, which were built around Excel and Python. In development, I discovered that most of the functionality needed could be accomplished, for better or worse, with just spreadsheet formulas. Keeping the data on Google Sheets rather than formatting it with BBCode, also allowed for user interaction to filter the available data.

StreamKit

StreamKit is a simple website to format posts on Stream, a fictional social network in the modern-tech Pacifica roleplay canon on the South Pacific forums.

StreamKit was built with Svelte and SvelteKit and is hosted on GitHub Pages.

Spyglass Archive

Spyglass Archive is a simple website that runs Spyglass once a day and saves the result. Spyglass itself was originally released here under the GPLv3 license. It generates Excel spreadsheets with estimated update times of each region.

The archive uses GitHub Actions to run Spyglass each day and saves the result to a site built with Jekyll and published with GitHub Pages.
